About Split Screen

“Split Screen” is a project I worked as part of Rensselaer Center for Open Source Software (RCOS). The goal of this program is to make window partitioning as simple as a click of a button. It would give the user authority to place their windows however they like, click a button, and generate a template of that certain window set up for reuse. The program also includes a number of base templates that users might find useful.

The images you see are images from one of the beginning iterations of UI/UX design. Since then, there have been many changes to how the program operates. That being said, I was recommended to put off future treatments until more of the functionality has been finalized.
